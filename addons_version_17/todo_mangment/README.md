# To Do App (`todo_mangment`)

A simple task management module for Odoo 17 with mail integration.

## Features

- Task creation and tracking
- Task views and menus
- Mail chatter integration via `mail` dependency

## Module Info

| Property | Value |
| --- | --- |
| Name | To Do App |
| Version | 1.0 |

| Website | http://www.yourcompany.com |

| Application | Yes |

## Data Files

- `security/ir.model.access.csv`
- `views/base_menu_view.xml`
- `views/todo_task_view.xml`

## Installation

1. Place this module in your Odoo addons directory.
2. Update the app list.
3. Install **To Do App**.
