# App One - Property Management Module

A property management module for Odoo 17 covering buildings, owners, sales, and accounting integration.

## Features

- Property and building management
- Owner and tag tracking
- Sales order integration
- Account move integration
- Property history tracking
- Printable reports
- Backend assets for list views and styling

- `web.report.assets_common`: report font CSS

## Installation

1. Place this module in your Odoo addons directory.
2. Update the app list.
3. Install **App one**.
