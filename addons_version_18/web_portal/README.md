# web_portal

Learning Odoo 18 Portal module.

## Features
- Portal templates for the website.
- OWL property form component with styles.

## Dependencies
- Odoo 18
- `base`
- `portal`
- `app_one`

## Assets
Frontend assets are loaded from `web.assets_frontend` and include Bootstrap 5 via CDN and the local OWL component files.

## Install
1. Add this addon to your Odoo addons path.
2. Update apps list.
3. Install `web_portal`.
