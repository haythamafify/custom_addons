Overview

HR Contract No End Date Alert is an Odoo 18 module that protects your organization from accidentally saving employee contracts without an end date. It displays a confirmation dialog before saving, automatically logs the decision to the contract's Chatter, and provides a quick search filter to audit all open-ended contracts at a glance.

✓ Odoo 18.0 ✓ Human Resources LGPL-3 Arabic & English v18.0.1.0.0

Key Features

Smart Save Alert

A confirmation dialog appears whenever a user tries to save a contract without an end date — preventing accidental open-ended contracts before they happen.

Chatter Tracking

The skip_end_date_check field is tracked (tracking=True). Every change is automatically logged in the contract's Chatter with user and timestamp.

Search Filter

A dedicated Contracts Without End Date filter is added to the contract search bar for instant visibility and auditing across all contracts.

Visual Highlight

The End Date field is visually highlighted (bold + warning color) when empty, giving an immediate at-a-glance signal directly in the form view.

Full i18n Support

All user-facing strings use _t() / _(). Arabic (ar_001.po) and English translation files are included out of the box.

Auto Chatter Message

When a contract is saved without an end date, a notification is automatically posted to the Chatter documenting the open-ended decision with full context.

How It Works

1
User opens or creates an HR contract — the module is fully transparent and adds no visible extra fields to the form.
2
User tries to save without filling End Date — a dialog immediately appears: ⚠ Contract End Date Missing — Do you want to continue saving?
3
If confirmed — the contract is saved, skip_end_date_check is set to True, and a Chatter notification is posted automatically.
4
Managers can filter — using the Contracts Without End Date search filter to audit all open-ended contracts across the organization.

Installation

Requires Odoo 18.0 with the hr_contract module installed.
  1. Copy the hr_contract_no_end_date_alert folder to your Odoo addons directory.
  2. Restart the Odoo server.
  3. Go to Apps → Update Apps List.
  4. Search for "HR Contract No End Date Alert" and click Install.

Configuration

No configuration required. The module works immediately after installation with default Odoo HR contracts.
