# Counter OWL Application

Counter OWL Application is an Odoo 18 module that demonstrates OWL components, services, and integrations through a set of small apps (counter, calculator, weather, CRM and sales helpers).

## Features
- Counter and calculator examples (classic and smart versions).
- OWL services demo (counter service, weather service, ORM service).
- Integrated workspace and todo environment examples.
- CRM lead creator and sales order lead tracker components.

## Dependencies
This module depends on:
- `base`
- `web`
- `crm`
- `sale`
- `stock`
- `stock_barcode`

## Installation
1. Add this repository to your Odoo addons path.
2. Update the apps list.
3. Install **Counter OWL Application** from Apps.

## Usage
After installation, open the module menu and launch the available OWL demo components from the provided menu entries.

## Assets
Backend assets are declared in `web.assets_backend` and live under `counter/static/src/`.
