# OWL Tutorial (Odoo 18)

Simple Todo List module built with OWL framework and Bootstrap 5.

## Features
- OWL-based Todo list UI
- Bootstrap 5 styling and icons
- Odoo backend views and models

## Requirements
- Odoo 18
- Apps: `base`, `web`, `mail`, `sale_management`

## Install
1. Add this module to your Odoo addons path.
2. Update the apps list.
3. Install **OWL Tutorial** from the Apps menu.

## Module Structure
- `controllers/` HTTP controllers
- `models/` Odoo models
- `views/` XML views and templates
- `static/src/components/` OWL components
- `security/` access control

## Assets
This module loads Bootstrap 5 CSS/JS and Bootstrap Icons via CDN through `web.assets_backend`.
